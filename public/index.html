🟡 Plex Setup
Configure StreamForge as a Live TV source in Plex
1
Import your Plex Library
Go to Libraries → Add Library → choose Plex. Enter your Plex server URL and token, then scan to import your media.
2
Build your channels and playout
Create channels in Channels, then schedule your Plex media in Playout. The EPG is automatically generated.
3
Add to Plex as a Live TV tuner
In Plex: Settings → Live TV & DVR → Set Up Plex DVR → Other Device → Add Manually
M3U Playlist URL
EPG / XMLTV URL
4
Get your Plex Token
Sign into plex.tv, open any media item in your browser, then append ?X-Plex-Token= to the URL — your token is in the URL. Or go to Settings → Account → click your avatar → "XML" to find it.
🟣 Jellyfin Setup
Configure StreamForge as a Live TV source in Jellyfin
1
Import your Jellyfin Library
Go to Libraries → Add Library → choose Jellyfin. Enter your Jellyfin server URL and API key.
2
Build channels and playout schedule
Create channels and assign your Jellyfin media to them. The XMLTV EPG is generated automatically from your schedule.
3
Add to Jellyfin Live TV
In Jellyfin: Dashboard → Live TV → + (Add Tuner Device) → M3U Tuner
M3U Playlist URL
Then add Guide Data Provider → XMLTV
4
Get your Jellyfin API Key
In Jellyfin: Dashboard → Advanced → API Keys → + (New API Key). Give it a name like "StreamForge" and copy the key.
